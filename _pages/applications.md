This page collects software demonstrations, simulation tools, open-source code, and videos related to Smart Mobility Lab research applications.

Simulation Tools
======

<div class="notice">
  <h3>Order Matching Simulation</h3>
  <p>The taxi market operates under two matching methods: dispatch and broadcast. This simulation tool lets users select the matching mode and upload driver and passenger request information to obtain matching results, which can be downloaded or displayed on a map.</p>
  <p><a class="btn btn--primary" href="https://www.civil.hku.hk/kejintao/softwares_simulation.html">Go to simulator</a></p>
</div>

<div class="notice">
  <h3>Large-scale Simulation Platform</h3>
  <p>This large-scale simulation platform supports the management and control of ride-hailing vehicles, including idle cruising, passenger pickup, and passenger delivery over large-scale transportation networks.</p>
  <p><a class="btn btn--primary" href="https://www.civil.hku.hk/kejintao/softwares_simulation.html">Go to demo</a></p>
</div>

Open-Source Code
======

- [Transportation Simulator](https://github.com/HKU-Smart-Mobility-Lab/Transportation_Simulator): an open-sourced traffic simulation platform for on-demand ride service operations.
- [Ride-sharing Simulator](https://github.com/HKU-Smart-Mobility-Lab/Ride-sharing-Simulator): an open-sourced simulator for high-capacity ride-sharing services.
- [AI-Code](https://github.com/hku-kejintao/AI-Code): coding practice tutorials for teaching and learning AI for civil engineering.
