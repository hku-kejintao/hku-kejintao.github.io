# Jintao Ke Personal Homepage

This repository hosts the personal academic homepage for Dr. Jintao Ke at:

```text
https://hku-kejintao.github.io/
```

The site is built with the [Academic Pages](https://github.com/academicpages/academicpages.github.io) Jekyll template and deployed through GitHub Pages.

## Main Files

- `_config.yml`: site-wide settings, author profile, social links, and GitHub Pages URL.
- `_data/navigation.yml`: top navigation menu.
- `_pages/about.md`: homepage biography.
- `_pages/research.md`: research areas and group links.
- `_pages/publications.html`: publications page.
- `_pages/teaching.html`: teaching page.
- `_pages/videos.md`: video page.
- `_pages/cv.md`: CV page.

## Local Preview

Install dependencies and run Jekyll locally:

```bash
bundle install
bundle exec jekyll serve
```

Then open:

```text
http://localhost:4000
```
